SOFTCLEANING - SITIO MOBILE FIRST

Archivos incluidos:
- index.html
- css/styles.css
- js/main.js
- assets/img/ imágenes de ejemplo en SVG

Cómo usar:
1. Abrí index.html en el navegador para ver la página.
2. Para cambiar el número de WhatsApp:
   - En index.html buscá: 549XXXXXXXXXX
   - En js/main.js cambiá la constante WHATSAPP_NUMBER.
3. Para cambiar precios:
   - En index.html buscá: Desde $____
4. Para cambiar imágenes:
   - Reemplazá los archivos dentro de assets/img/
   - Si usás JPG o PNG, cambiá el nombre en index.html.
5. Para subirlo a un hosting:
   - Subí todos los archivos y carpetas manteniendo la misma estructura.

Recomendación:
Usá imágenes reales de trabajos: limpieza interna, antes/después, notebooks abiertas, escritorio de reparación.
Eso vende mucho más que imágenes genéricas.


Cambios de esta versión:
- Nombre agregado: SoftCleaning.
- Planes en celular con desplazamiento horizontal.
- Hero corregido para evitar superposición visual en móvil.
